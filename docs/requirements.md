# knitLog 요구사항 정의서

## 1. 서비스 개요
KnitLog는 뜨개질 작품의 위시리슽, 진행중인 프로젝트, 게이지 계산, 실 재고 및 바늘 정보를 관리하는 퍼스널 니팅 관리 서비스이다.

## 2. 주요 기능
### 2.1 Wish
뜨고 싶은 작품을 저장하고 관리한다.
    
    ### 기능
    - 작품 등록
    - 작품 조회
    - 작품 수정
    - 작품 삭제
    - Wish 정보를 기반으로 Knitting Project 생성
    
    ### 저장 정보
    - 제작자
    - 작품명
    - 도안 URL(다중)
    - 원작 실
    - 원작 바늘 호수
    - 원작 게이지
    - 메모
    
    ### 비고
    - Wish를 기반으로 Knitting Project 생성 가능
    - Project 생성 후에는 Wish 수정과 독립적으로 관리

### 2.2 Knitting Project
진행 중인 작품을 프로젝트 단위로 관리한다.
    
    #### 상태
    - CO: 시작
    - WIP: 진행 중
    - HOLD: 중단/보류
    - FO: 완성
    
    #### 기능
    - 프로젝트 등록
    - 프로젝트 조회
    - 프로젝트 수정
    - 프로젝트 삭제
    - Wish에서 정보 불러오기
    - 저장된 Gauge 불러오기
    
    #### 저장 항목
    - 작품명
    - 상태
    - 제작자
    - 도안 URL 목록
    - 원작 실 정보
    - 사용 실 정보
    - 원작 바늘 호수
    - 사용 바늘
    - 원작 게이지
    - 사용 게이지
    - CO 정보
    - 메모

### 2.3 Knitting Log
하나의 Knitting Project에 여러 개의 진행 기록을 저장한다.

    #### 기능
    - 진행 기록 등록
    - 진행 기록 조회
    - 진행 기록 수정
    - 진행 기록 삭제
    
    #### 저장 항목
    - 진행 코멘트
    - 변경사항
    - 진행 기록 사진
    - 작성일


### 2.4 Gauge
실과 바늘 조합에 따른 게이지를 저장하고, 도안 게이지와 내 게이지를 비교하여 변환 계산을 지원한다.

    #### 기능
    - 게이지 계산
    - 게이지 저장
    - 게이지 조회
    - 게이지 수정
    - 게이지 삭제
    - Knitting Project에서 저장된 게이지 불러오기
    
    #### 저장 항목
    - 실 정보
    - 바늘 정보
    - 10cm 기준 코수
    - 10cm 기준 단수
    - 메모

### 2.5 Yarn
실 정보와 실 재고를 분리하여 관리한다.

    #### 기능
    - 실 정보 등록
    - 실 정보 조회
    - 실 정보 수정
    - 실 정보 삭제
    - 실 재고 등록
    - 실 재고 수정
    - 실 재고 차감
    
    #### 저장 항목
      ##### Yarn Info
      - 브랜드
      - 제품명
      - 계절
      - 실 종류
      - 색상
      - 두께
      - 100g당 길이
      - g당 길이
      - 메모
    
      ##### Yarn Stock
      - 총 보유 g
      - 남은 g
      - 추가 g
      - 구매일
      - 메모

### 2.6 Needle
보유한 바늘 및 케이블 정보를 관리한다.

    #### 기능
    - 바늘 등록
    - 바늘 조회
    - 바늘 수정
    - 바늘 삭제
    - 케이블 등록
    - 케이블 조회
    - 케이블 수정
    - 케이블 삭제

    #### 저장 항목
      ##### Needle
      - 브랜드
      - 종류
      - mm
      - 길이
      - 메모
    
      ##### Needle Cable
      - 브랜드
      - 종류
      - 길이
      - 메모
